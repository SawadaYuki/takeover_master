00031  * The goal of including this file is to define most of the external
00032  *   symbols used in mtcp_sharetemp.c .  This to insure that the linker
00033  *   does not try to resolve any symbols by linking in libc.  That would
00034  *   fail at restart time, when there is no libc.

00036  *   of other files, allowing us to compile a single file into a static

00045  * Note that /usr/include/asm/unistd.h defines syscall using either:
